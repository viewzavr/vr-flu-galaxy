# vr-flu-galaxy

An interactive 3D visualization of the evolutionary paths.

![](vr.jpg)

Start: https://viewzavr.com/apps/vr-flu-galaxy

# input file format

Json-formatted file of trajectories in form:
```
 [
   [x,y,z,x,y,z,x,y,z,.....],
   [x,y,z,x,y,z,x,y,z,.....],   
   ....
   [x,y,z,x,y,z,x,y,z,.....],   
 ]   
```
